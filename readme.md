# Esercizio in C che permette di calcolare il quadrato di numeri pari
* Tipi di variabili:
+ i: contatore del for (intero);
+ num: numeri inseriti da utente (intero);
+ quadrato: quadrato di ogni numero pari (intero);
* Obiettivo
+ Visualizzare il quadrato di ogni numero pari
Algoritmo:
+ Utente: Inserisci un numero;
+ Input: stampa num;
se i mod 2 == 0
+ stampa "quadrato = i^2";
